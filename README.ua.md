# Практичні завдання з JavaScript – 14 вправ

Цей репозиторій містить 14 практичних завдань для тренування навичок роботи зі скриптами на JavaScript.  
Всі завдання спрямовані на роботу з DOM, подіями, інтерфейсом користувача та базовою логікою.

---

## Короткий опис завдань

1. **Зміна значення input** – при кліку на кнопку змінюється значення в полі введення.
2. **Alert з введеним значенням** – при кліку на кнопку показується alert з текстом з input.
3. **Обмін значеннями інпутів** – при натисканні кнопки два input обмінюються значеннями.
4. **Зміна тексту параграфів** – при натисканні кнопки текст абзаців змінюється на їх порядковий номер.
5. **Квадрат числа** – введене число підноситься до квадрату; при неправильному введенні показується alert.
6. **Створення списку через prompt** – користувач динамічно додає пункти у список до натискання ESC.
7. **Подія на посиланнях** – при наведенні на посилання його текст записується в атрибут title.
8. **Чергування кольору span** – по кліку на span його фон чергується між червоним та зеленим.
9. **Додавання пунктів в ol та делегування** – додаються нові елементи, працює делегування для обробки кліків.
10. **Блокування та розблокування input** – одна кнопка блокує поле, інша – розблоковує.
11. **Додавання двох чисел** – введені числа сумуються і результат показується у span.
12. **Втрата фокусу input** – при втраті фокусу input його значення виводиться у абзац.
13. **Вивід коду клавіші** – при натисканні клавіші показується її код.
14. **Вивід значення клавіші** – при натисканні клавіші показується сама літера або цифра.

---

Ці вправи допомагають навчитися:

- взаємодії з DOM та елементами сторінки,
- обробці подій користувача,
- створенню динамічного контенту,
- делегуванню подій та управлінню станом елементів,
- роботі з клавіатурою та базовою логікою в JavaScript.
