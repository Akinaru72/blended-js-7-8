# JavaScript Practical Exercises – 14 Tasks

This repository contains 14 practical exercises to practice JavaScript scripting skills.  
All tasks are focused on working with the DOM, events, user interface, and basic logic.

---

## Task Overview

1. **Change input value** – clicking a button changes the value of an input field.
2. **Alert with input value** – clicking a button shows an alert with the value from the input.
3. **Swap input values** – clicking a button swaps the values of two inputs.
4. **Change paragraph text** – clicking a button replaces paragraph text with its index number.
5. **Square a number** – the entered number is squared; if input is invalid, an alert is shown.
6. **Create a list via prompt** – the user dynamically adds items to a list until ESC is pressed.
7. **Event on links** – on hover, the link’s text is saved to its `title` attribute.
8. **Toggle span color** – clicking a span alternates its background color between red and green.
9. **Add items to ol & delegation** – new list items are added, click events use delegation.
10. **Lock and unlock input** – one button disables the input, another enables it.
11. **Add two numbers** – the sum of two inputs is displayed in a span.
12. **Input blur event** – when an input loses focus, its value is displayed in a paragraph.
13. **Show key code** – pressing a key displays its code in a span.
14. **Show key value** – pressing a key displays the actual letter or number in a span.

---

These exercises help you learn:

- interacting with the DOM and page elements,
- handling user events,
- creating dynamic content,
- event delegation and element state management,
- working with the keyboard and basic JavaScript logic.
